## Overview

Aptible offers a range of powerful integrations to enhance observability, security, and the development workflow for applications on the platform. Here's a glimpse of some of the key integration categories available on Aptible.

---

### Databases

Aptible offers a robust selection of managed databases, including CouchDB, Elasticsearch, InfluxDB, MongoDB, PostgreSQL, RabbitMQ, Redis, and SFTP. [Learn more →](/docs/databases)

---

### Observability

#### Log Drains

Log Drains allow you to centralize and analyze application logs. By integrating with popular log analysis tools, you can gain valuable insights into your application's performance and troubleshoot issues effectively. Aptible supports a number of log drain integrations, such as Amazon S3, Datadog, Elasticsearch, HTTPS, Logentries, LogDNA, Logstash, Papertrail, Sumo Logic, Syslog, and more. [Learn more →](/docs/databases)

#### Metric Drains

Metric Drains enable you to export application metrics to external monitoring and analytics services. This integration enhances your ability to monitor and optimize application performance. Aptible supports Datadog and InfluxDB metric drains. [Learn more →](/docs/databases)

---

### Developer tools

#### CI/CD Integrations

Aptible integrates seamlessly popular Continuous Integration/Continuous Deployment (CI/CD) tools. Automate your application deployment pipeline for faster and more reliable releases. [Learn more →](/docs/continuous-integration-provider-deployment)

#### Terraform

Aptible supports Terraform, a powerful Infrastructure as Code (IaC) tool. Use Terraform to define and provision your infrastructure on Aptible, ensuring consistency and reproducibility. [Learn more →](/docs/terraform)

---

### Network

#### VPNs

Aptible supports site-to-site Virtual Private Networks (VPNs), providing secure and private network connections between your infrastructure and external resources. [Learn more →](/docs/network-integrations)

#### VPC Peering

VPC Peering enables secure communication between different Virtual Private Clouds (VPCs). This integration is ideal for connecting isolated network environments and extending your infrastructure. [Learn more →](/docs/network-integrations)

---

### Security

#### Twingate

Twingate integration provides a robust Zero Trust Network Access solution. Securely access your Aptible resources from anywhere while ensuring strong authentication and granular access control. [Learn more →](/docs/twingate)

#### SAML

Security Assertion Markup Language (SAML) integration enhances your application's security by enabling Single Sign-On (SSO) authentication. Securely manage user access and streamline authentication processes with this integration. [Learn more →](/docs/sso)

#### Sym

Sym is a tool that provides just-in-time Slack-based approvals for access requests to Aptible resources. Enhance security by implementing robust approval workflows for resource access. [Learn more →](/docs/Sym)
